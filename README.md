##### Tipos de Datos:

* Enteros (int): Representan n√∫meros enteros. Ejemplos: 5, -10, 1000.


* N√∫meros de Punto Flotante (float): Representan n√∫meros decimales o de punto flotante. Ejemplos: 3.14, -0.01, 2.0.

* Cadenas de Texto (str): Representan secuencias de caracteres. Ejemplos: 'Hola, mundo', "Python es genial".

* Booleanos (bool): Representan valores de verdad, ya sea True o False. Se utilizan en expresiones l√≥gicas y de control de flujo.

* Listas (list): Son secuencias ordenadas y modificables de elementos. Pueden contener cualquier tipo de dato, incluso otras listas. Ejemplo: [1, 2, 'a', [4, 5]].

* Tuplas (tuple): Son secuencias ordenadas e inmutables de elementos. Se definen utilizando par√©ntesis. Ejemplo: (1, 2, 'b').

* Conjuntos (set): Representan colecciones no ordenadas y sin duplicados. Se definen utilizando llaves. Ejemplo: {1, 2, 3}.

* Diccionarios (dict): Asocian claves con valores y son √∫tiles para representar mapeos. Se definen utilizando llaves y dos puntos. Ejemplo: {'nombre': 'Juan', 'edad': 25}.
